# Contributing to Faces of Open Source Uzbekistan

Thank you for your interest in contributing! This project celebrates the Uzbek open source community.

## Ways to Contribute

### 1. Submit a Nomination

Visit [facesofopensource.uz/submit.html](https://facesofopensource.uz/submit.html) to nominate someone for the project.

### 2. Improve the Website

Found a bug or have an enhancement? Here's how to contribute code:

1. **Fork the repository**
2. **Create a feature branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes**
4. **Test locally**
   ```bash
   python3 -m http.server 8000
   ```
5. **Commit your changes**
   ```bash
   git commit -m "Description of changes"
   ```
6. **Push to your fork**
   ```bash
   git push origin feature/your-feature-name
   ```
7. **Create a Pull Request**

### 3. Add Content

To add a new person to the site:

1. Add their portrait to `media/portraits/`
2. Update `data/people.json` with their information
3. Submit a PR with both files

### 4. Translate

Help make the site accessible in multiple languages:
- Uzbek (Latin and Cyrillic)
- Russian
- English

### 5. Spread the Word

Share the project within the Uzbek tech community!

## Code Style

- Use consistent indentation (2 spaces)
- Write semantic HTML
- Keep CSS organized and commented
- Write clear commit messages
- Test on multiple browsers
