# Media Guidelines

## Portrait Photography

All photos stored in this directory should follow these guidelines:

### Technical Requirements
- **Format:** JPG
- **Size:** 1200x1600px minimum (3:4 aspect ratio)
- **Quality:** High resolution, at least 300 DPI for print
- **Style:** Black and white or easily convertible to grayscale
- **File naming:** `firstname-lastname.jpg` (lowercase, hyphenated)

### Photography Style
Following the Faces of Open Source aesthetic:
- Clean, simple backgrounds
- Good lighting that highlights facial features
- Neutral or professional expression
- Subject looking at camera or slightly off-camera
- Head and shoulders composition

### File Size
- Optimize images for web (keep under 500KB if possible)
- Use tools like ImageOptim, TinyPNG, or similar
- Balance quality and performance

## Adding Photos

1. Place original high-res photo here: `media/portraits/`
2. Optimize for web
3. Update `data/people.json` with the path
4. Commit and push to repository

## Example

```json
{
  "image": "media/portraits/john-doe.jpg"
}
```
